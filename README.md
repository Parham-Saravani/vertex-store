## About The Project

Vertex Store is a modern Full Stack E-Commerce platform developed using Vanilla JavaScript, Express.js and MongoDB.

The project provides a responsive shopping experience with category filtering, product management, dark mode support, custom REST APIs and a scalable architecture for future development.

---

## Features

### Frontend

* Responsive Design
* Modern User Interface
* Dark / Light Mode
* SPA-like Navigation
* Reusable Components
* Custom Product Cards

### Backend

* RESTful API Architecture
* Product CRUD Operations
* MongoDB Integration
* Mongoose Models
* Route & Controller Pattern
* Modular Project Structure


## API Endpoints

### Products

```http
GET    /api/products
POST   /api/products
```
### Messages

```http
POST   /api/messages
```
### Users

```http
GET    /api/users
```

## Installation

### Clone Repository

```bash
git clone https://github.com/Parham-Saravani/vertex-store.git
```

### Frontend

```bash
cd frontend

npm install

npm run dev
```

### Backend

```bash
cd backend

npm install

npm run dev
```
